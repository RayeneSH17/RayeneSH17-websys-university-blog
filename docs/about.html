Project Background

This website was developed as part of the Web Systems course at Warsaw Management University. The goal of the project is to practice creating a simple multi-page website using HTML, CSS, and GitHub Pages for hosting.

What this blog demonstrates

The blog includes: page structure, navigation, images, external CSS, and a clean academic layout.

In a real scenario, this blog could be used to share updates about student projects, events, and academic news.
